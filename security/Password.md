# 密码管理

- [KeepPass](https://keepass.info)
- LastPass
- 1Password

## KeepPass

- Mac

    1. 安装客户端 [KeeWeb](https://keeweb.info/)
    2. 使用 KeeWeb 新建一个密码文件并上传至 OneDrive
    3. 使用 KeeWeb 以 OneDrive 的形式打开云同步的密码文件

- Android

    1. 安装客户端 Keepass2Android
    2. 使用 Keepass2Android 以 OneDrive 的形式打开云同步的密码文件
