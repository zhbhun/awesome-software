- [Balance: Mindful time tracking](https://alexandersandberg.com/apps/balance/)

    这是一个 Mac 管理工作与生活平衡的时间跟踪应用程序。也可以拿来当番茄钟使用，制作非常精美，订阅付费的形式，很适合独立开发者做的工具类产品。

- [Opal: Screen Time for Focus](https://apps.apple.com/app/id1497465230?mt=8)

    这是一个 iOS 上的屏幕时间管理应用，做的非常不错，可以管理你手机上 APP 使用的时间，防止沉迷手机，这个产品上线有两年多了，看到下载量有 30 万，基本都是美国用户，订阅的变现模式，和谜底团队出品的 Offscreen 类似的产品。
